# Packages

This workspace contains reusable libraries that make up the SynapseIQ product substrate.

Expected package lanes:
- `schemas/`
- `contracts/`
- `ontology/`
- `normalization/`
- `transformation/`
- `enrichment/`
- `reasoning/`
- `grammars/`
- `lsp/`
- `utils/`

Each package should remain focused and independently testable.
